# Calibration Control System

Automated calibration instrument management and analysis system developed with Excel VBA.

---

## Overview

This project was developed to optimize the control and monitoring of calibration instruments through automation and intelligent data organization.

The system improves operational efficiency by reducing manual work and accelerating the identification of instruments approaching calibration expiration.

---

## Problem

The analyst previously used a highly manual spreadsheet containing only the “Master List” worksheet.

To identify instruments approaching calibration expiration, it was necessary to filter records one by one, which could take several hours to locate valid instruments within a specific date range.

Separating instruments by responsible personnel was also a time-consuming task, reducing operational efficiency and increasing manual workload.

---

## Proposed Solution

Using VBA automation, a smarter and more efficient system was developed while keeping the original “Master List” structure unchanged, according to the analyst’s preference.

With only a few clicks, the system automatically organizes and separates instruments, making the process significantly faster and more practical.

Tasks that previously required hours can now be completed in just a few minutes, greatly improving productivity and reducing manual effort in the calibration management process.

---

# Sistema de Controle de Calibração

Sistema automatizado de gestão e análise de instrumentos de calibração desenvolvido com Excel VBA.

---

## Visão Geral

Este projeto foi desenvolvido para otimizar o controle e o monitoramento de instrumentos de calibração por meio de automação e organização inteligente de dados.

O sistema melhora a eficiência operacional ao reduzir o trabalho manual e acelerar a identificação de instrumentos próximos do vencimento da calibração.

---

## Problema

O analista anteriormente utilizava uma planilha altamente manual contendo apenas a aba “Master List”.

Para identificar instrumentos próximos do vencimento da calibração, era necessário filtrar os registros um por um, o que podia levar várias horas para localizar instrumentos válidos dentro de um intervalo específico de datas.

A separação de instrumentos por responsável também era uma tarefa demorada, reduzindo a eficiência operacional e aumentando a carga de trabalho manual.

---

## Solução Proposta

Utilizando automação em VBA, foi desenvolvido um sistema mais inteligente e eficiente, mantendo a estrutura original da aba “Master List” inalterada, conforme preferência do analista.

Com apenas alguns cliques, o sistema organiza e separa automaticamente os instrumentos, tornando o processo significativamente mais rápido e prático.

Tarefas que antes levavam horas agora podem ser concluídas em poucos minutos, melhorando consideravelmente a produtividade e reduzindo o esforço manual no processo de gestão de calibração.

---

## Principais Funcionalidades

- Controle automatizado de calibração  
- Filtragem de vencimento de instrumentos  
- Organização automática por responsável  
- Separação inteligente de dados

- Processos de automação em VBA  
- Fluxo de análise mais rápido  
- Interface amigável ao usuário
